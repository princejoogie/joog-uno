URL Shortener with NextJS and TailwindCSS

## Installation

```bash
git clone --depth=1 https://github.com/princejoogie/joog-uno.git
cd joog-uno
rm -rf .git
git init .
```

## Usage

```bash
yarn dev # or npm run dev
```
